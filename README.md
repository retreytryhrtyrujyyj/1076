# 基于协同过滤算法商品推荐系统

## 项目简介
基于协同过滤算法的商品推荐系统及基于Java语言，Springboot框架，Mysql数据库， HTML/CSS/JavaScript的商品推荐系统，采用协同过滤算法计算推荐用户可能喜欢的其他相似的商品。适合作为毕业设计、大作业、课程设计等。

Java;SpringBoot;Mysql

## 开发工具
1.IntelliJ IDEA或者Myeclipse 3.Navicat 3.Mysql 4.JDK8

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
